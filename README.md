# SpringBoot + H2 with Caffeine Cache


# Getting Started

## curl commands

```bash
curl --location --request POST 'http://localhost:8080/employees' \
--header 'Content-Type: application/json' \
--data-raw '{
    "name": "bike"
}'
```

```bash
curl --location --request GET 'http://localhost:8080/employees/1' \
--data-raw ''
```
